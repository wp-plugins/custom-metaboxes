Create both custom metaboxes and post types for your wordpress application.

== Description ==
Using this plugin will allow you to create entirely new sections in your wordpress application quick. With this plugin you can create new fields for your current post types such as the pages section or posts section. You can also create new sections along with new fields for those sections.

The steps are:

1. Activate Plugin
2. Click on the MetaBoxes section on the left navigation.
3. Click on the Add New Button next to the section you wish to manage.
4. Fill in the on screen fields and follow the on screen instructions to complete it.


== Installation ==

1. Upload `custom-metaboxes` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How can I use this plugin? =

After activating this plugin you will see a new menu item appear to your left called MetaBoxes where you can then start managing your custom metaboxes / fields and post types.

= What versions do you support? =

So far I have only tested in version 3.0+
